# **Atoll Network Planning Software**

**Atoll** is a multi-technology radio frequency planning and optimization platform used by telecom operators for wireless network design throughout the lifecycle.

Atoll typically supports:
- 2G/3G/4G/5G radio network planning and optimization.
- Automatic cell planning, coverage prediction, and capacity analysis.
- Monte Carlo simulations and multi-RAT network modeling.
- GIS integration for site selection and propagation modeling.

## License and Cost

Commercial software with enterprise licensing for telecom operators.

## Limitations

High complexity requires specialized RF engineering expertise.

## Useful References

- **[Atoll Official Website](https://www.forsk.com/atoll)**
- **[Atoll Documentation](https://www.forsk.com/resources)**
- **[Forsk Support Portal](https://support.forsk.com)**

# **NetSim Telecommunications Simulator**

**NetSim** is a network simulation software specializing in wired and wireless telecom technologies including 5G, WiFi, and IoT networks.

NetSim typically supports:
- End-to-end 5G NR, LTE, WiMAX, and MANET simulations.
- Protocol modeling at OSI layers 1-7 with configurable parameters.
- Wireless propagation models and mobility scenarios.
- GUI-based experimentation with integrated analytics.

## License and Cost

Commercial academic and professional editions with volume licensing.

## Limitations

Simulation speed limitations for very large-scale networks.

# **OMNeT++ Network Simulation Framework**

**OMNeT++** is an extensible, component-based C++ simulation library widely used for modeling complex telecom and communication networks.

OMNeT++ typically supports:
- Modular framework for discrete event simulation.
- INET framework for TCP/IP, wireless, and mobile networks.
- 5G/6G modules, vehicular networks, and IoT simulations.
- Hierarchical model composition and parallel execution.

## License and Cost

Open-source under Academic Public License; commercial Eclipse license available.

## Limitations

Requires C++/programming knowledge for custom model development.

## Useful References

- **[OMNeT++ Official Website](https://omnetpp.org)**
- **[INET Framework](https://inet.omnetpp.org)**
- **[OMNeT++ Community](https://omnetpp.org/community)**

# **NS-3 Network Simulator**

**NS-3** is a discrete-event network simulator for Internet systems, focused on research and education in wireless and wired telecom protocols.

NS-3 typically supports:
- LTE/5G NR, WiFi 6, WiMAX, and satellite communication modules.
- Realistic radio propagation and channel models.
- Real-time emulation and distributed simulation capabilities.
- Extensive protocol implementations from TCP to application layer.

## License and Cost

Completely free and open-source under GNU GPLv2 license.

# **Simu5G Network Simulator**

**Simu5G** is an open-source OMNeT++-based simulator specifically designed for modeling and simulating 5G New Radio (NR) networks with realistic protocol stacks.

Simu5G typically supports:
- Complete 5G NR protocol stack from PHY to application layers.
- Standalone (SA) and ENDC deployments with 4G/5G coexistence.
- Macro/micro/pico gNBs, carrier aggregation, and handover support.
- MEC modeling, D2D communications, and real-time emulation capabilities.

## License and Cost

Free open-source under Academic Public License.

## Limitations

Requires OMNeT++ framework knowledge for advanced customizations.

## Useful References

- **[Simu5G Official Website](https://simu5g.org)**
- **[Simu5G Documentation](https://simu5g.org/docs)**
- **[OMNeT++ Tutorials](https://omnetpp.org/doc)**

# **MATLAB 5G Toolbox**

**MATLAB 5G Toolbox** provides standard-compliant algorithms for link-level and system-level simulation of 5G NR and 5G-Advanced communications systems.

MATLAB 5G Toolbox typically supports:
- Waveform generation, channel modeling, and end-to-end link simulation.
- CDL/TDL channel models for terrestrial and NTN scenarios.
- Block error rate, throughput analysis, and golden reference verification.
- Customizable reference designs for 6G technology exploration.

## License and Cost

Commercial MathWorks licensing with academic and volume options.

## Limitations

High cost; MATLAB programming required for complex scenarios.
